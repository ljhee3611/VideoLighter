# VideoLighter v1.0.0 정식 서비스 상세 기술 명세서 (Service Overview)

## 1. 개요 (Introduction)
**VideoLighter**는 일반 사용자가 영상 인코딩 기술을 몰라도, 클릭 한 번으로 원본급 화질을 유지하면서 용량만 획기적으로 줄여주는 **데스크탑 전용 동영상 최적화 솔루션**입니다. 클라우드 방식이 아닌 사용자 PC의 리소스를 활용하는 100% 오프라인 처리 방식을 채택하여 개인정보 보호와 처리 속도를 모두 잡았습니다.

---

## 2. 핵심 기술 (Core Technology)

### 💎 인지 화질 최적화 (Perceptual Quality Optimization - Subjective VQ)
*   **원리:** 기계적인 SSIM/PSNR 수치에만 매몰되지 않고, 인간의 시각 시스템(HVS)이 민감하게 반응하는 영역(얼굴, 자막, 밝은 피사체 등)에 비트레이트를 집중 배정합니다.
*   **효과:** 배경이나 어두운 영역 등 덜 중요한 데이터는 과감히 깎아내어, 육안으로는 차이를 느끼지 못하면서 용량은 기존 인코딩 대비 추가로 20~30% 더 절감합니다.

### 🌈 10-bit HDR (High Dynamic Range) 보존
*   **문제 해결:** 시중의 많은 압축기들이 아이폰/갤럭시 최신폰으로 찍은 HDR 영상을 처리할 때 색감이 물 빠진 것처럼 변하는 현상이 발생합니다.
*   **기술력:** VideoLighter는 10-bit 색상 심도와 HDR 메타데이터를 그대로 파이프라인에 태워 전송하므로, 쨍하고 선명한 원본의 색 선명도를 100% 유지합니다.

### 💨 고효율 차세대 코덱 기본 채택 (AV1 & VP9)
*   **AV1:** 최신 하이테크 코덱으로, 기존 H.264(MP4) 대비 동일 화질에서 최대 50% 이상의 압축률을 자랑합니다.
*   **VP9:** AV1을 지원하지 않는 기기에서도 최상의 화질을 보장하기 위해 범용적이고 강력한 대안으로 사용됩니다.

## 3. 주요 기능 실무 가이드 (Feature Breakdown)

### 🥇 3단계 황금 프리셋 (Golden Presets)
사용자의 상황에 맞춰 인코딩 엔지니어가 직접 튜닝한 세팅값을 버튼 하나에 담았습니다.
1.  **최고 화질 (Level 4):** 비손실에 가까운 인코딩. 용량은 30~50% 줄지만 전문 장비로 대조해도 구분이 불가능합니다.
2.  **밸런스 모드 (Level 6):** 웹 배포 및 SNS 공유에 최적화. 육안상 완벽하며 용량을 절반 이하로 줄입니다. (기본 설정)
3.  **용량 우선 (Level 8):** 카카오톡, 이메일 등 전송 용량 제한이 엄격할 때 사용합니다. 용량 다이어트를 극한으로 수행합니다.

### 🛡️ 개인정보 보호 및 보안 (Privacy & Security)
1.  **100% 오프라인 처리:** 단 1Byte의 영상 데이터도 서버로 전송하지 않습니다. 인터넷 연결 없이도 작동합니다.
2.  **메타데이터 클리닝 (Optional):** 영상 파일 속에 숨어있는 GPS 위치 정보, 촬영 날짜, 기기 일련번호 등 민감한 개인정보를 선택적으로 완벽하게 도려낼 수 있습니다.

### 🚀 스마트 성능 최적화 (Performance)
*   **병렬 처리 시스템:** CPU 코어 수에 맞춰 여러 개의 영상을 동시에 변환합니다. (기본 2개 설정으로 시스템 버벅임 최소화)
*   **Turbo Acceleration:** 고사양 하드웨어 가속을 활용하여 인코딩 속도를 한계까지 끌어올립니다.

---

## 4. 사용자 인터페이스 디테일 (UX/UI)
*   **드래그 앤 드롭:** 파일 하나부터 수백 개의 파일이 든 폴더까지 그냥 던져넣기만 하면 자동으로 목록을 구성합니다.
*   **출력 경로 자동 설정:** '원본과 같은 경로' 옵션을 통해 처리된 파일을 찾기 위해 헤맬 필요가 없습니다. (기존 원본을 휴지통으로 바로 보내는 기능 포함)
*   **진행 상태 직관화:** 전체 진행률과 현재 파일의 상태를 미려한 애니메이션으로 실시간 확인할 수 있습니다.

---

## 5. 비즈니스 가치 (User Benefits)
*   **구독료 제로:** 한 번 구매 시 평생 업데이트 및 사용을 보장합니다.
*   **비용 절감:** 아이클라우드, 구글 포토 등 클라우드 스토리지 추가 결제를 늦춰줍니다.
*   **시간 절감:** 대용량 영상 업로드/전송 시간을 비약적으로 단축시킵니다.

---

## 6. 권장 사양 (System Requirements)
*   **OS:** Windows 10 / 11 (64-bit)
*   **CPU:** Intel Core i3 10세대 이상 / AMD Ryzen 3 3세대 이상 (멀티코어 권장)
*   **RAM:** 8GB 이상 권장
*   **기타:** FFmpeg Library 기반 (Tauri Sidecar 통합 방식)
